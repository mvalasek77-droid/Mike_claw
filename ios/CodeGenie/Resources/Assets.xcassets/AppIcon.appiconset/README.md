# App Icon

Drop a `icon-1024.png` here.

- 1024 × 1024 pixels
- sRGB color space
- **no alpha channel** (Apple rejects icons with alpha)
- **no rounded corners** (Apple applies the mask)

The fastest way to produce this is the in-app **Icon Forge** screen:
Home → BitDrop neighbour tile → Icon Forge, or from any build's success
screen. It will export an App-Store-safe PNG that drops straight here.
